Search Tips

The search bar supports both fuzzy matching and advanced filter operators.

Fuzzy Search

Just type any part of a file name and the search engine will find approximate matches. For example, typing samsung will find all Samsung firmware files.

Advanced Operators

Use these operators in the search bar for precise filtering:

Operator Example Description
brand:brand:SamsungFilter files by brand
ext:ext:zipFilter files by extension
type:type:archiveFilter files by type

You can combine operators: brand:Samsung ext:zip Android

Filtering & Sorting

Sidebar Filters

Click on any brand or file type in the left sidebar to filter the file list. The count badge shows how many files match each filter.

Advanced Filters

Click the icon next to the search bar to open the advanced filter panel. You can filter by extension, brand, or file name.

Sorting

Click the sort icon in the header to sort files by name, extension, or brand in ascending or descending order. In list view, you can also click column headers to sort.

View Modes

Grid View

Shows files as visual cards with large icons, file name, brand badge, and extension. Hover over a card to reveal the download button. Press g to activate.

List View

Shows files in a compact table format with columns for name, brand, type, and download action. Press l to activate.

Themes

GetwayROM supports both light and dark themes. Click the moon icon in the header or press t to toggle. Your preference is saved automatically.
